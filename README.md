# Pixel Legends – Frontend UI Prototype

## Overview

Pixel Legends is a frontend web interface prototype developed during the **Sastranet Bootcamp Project Day**.
The project focuses on building a visually engaging landing page for a fictional character collection game.

The page includes a hero section with a background video, navigation bar, and character showcase cards.

## Features

* Hero section with video background
* Navigation bar with action buttons
* Character showcase cards
* Custom fonts and styled UI components
* Modern landing page layout

## Technologies Used

* HTML5
* CSS3
* Google Fonts

## Project Structure

index.html – Main webpage structure
style.css – Styling and layout
assets/ – Images, video, and fonts used in the project

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in any web browser

## Learning Outcome

This project was developed as part of the **Sastranet Bootcamp** to practice frontend development fundamentals including layout design, styling, and UI structuring using HTML and CSS.

Future improvements include adding JavaScript for interactivity and responsiveness.
